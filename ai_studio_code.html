多功能网关连接示意图

多功能采集单元（MCU）具备强大的兼容性，支持混合接入多种传感器类型，满足复杂工程场景需求：

  • 振弦类传感器：全面支持钢筋计、轴力计、渗压计、土压力盒、应变计、锚索测力计等，内置高精度扫频激励算法，确保数据精准。
  • 数字/模拟信号：支持 RS485 (Modbus)、4-20mA、0-5V 等通用接口设备接入。全面支持测距仪、倾角仪、静力水准等数字信号传感器。
  • 统一传输：所有数据通过 MCU 汇聚，利用 4G/LoRa 网络实时上传至云端，实现“端-边-云”高效协同。

前端监测平台

基于 Next.js + React 构建的 Web 前端监测平台，为基坑工程提供项目级总览、设备告警联动、地图化监测点与报表输出，一站式服务业主与监测单位。

  • dashboard 项目安全总览：集中展示项目核心指标，支持在线管理。
  • sensors 设备告警联动：实时统计设备状态，多维度查看告警信息。
  • map 地图化监测：基于 GIS 地图，直观展示监测点状态与关键数据。
  • insights 多维度分析：集成 ECharts，提供专业的数据趋势曲线与风险评估。
  • chat 内置 AI 助手：自然语言交互，快速查询设备状态与历史趋势。
