前端监测平台

基于 Next.js + React 构建的 Web 前端监测平台，为基坑工程提供项目级总览、设备告警联动、地图化监测点与报表输出，一站式服务业主与监测单位。

  • dashboard 项目级安全总览：首页仪表盘集中展示项目进度、基坑深度、关键责任人信息，并支持在线编辑项目信息。
  • sensors 设备与告警联动：与设备管理、告警中心打通，统计在线/离线/告警设备数量，结合时间维度查看近期告警。
  • map 地图化监测点总览：基于天地图与 OpenLayers，在地图上按状态着色监测点，悬停即可查看关键遥测数据与异常原因。
  • insights 多维度趋势分析：集成 ECharts，对位移、倾角等多参数按时间窗口进行趋势曲线展示，为风险评估提供数据支撑。
  • chat 内置 AI 助手：支持在前端界面侧边栏呼出 AI 助手，以自然语言查询设备状态、告警原因与历史趋势。

AI 驱动的自动化监测

系统不仅仅是数据的搬运工。内置的 AI 分析引擎让平台具备了“思考”能力，实现从被动监测到主动预警的跨越。

  • trending_up 智能趋势预测：基于历史数据自动外推变形趋势。
  • account_tree 自动化工作流：配置“超限→AI 复核→告警”全流程。
  • chat 对话式助手：自然语言查询监测数据与状态。
  • assignment 自动报告生成：一键生成日报/周报草稿。
